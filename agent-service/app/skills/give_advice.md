# Skill: Give Advice

## What this skill does
Provides specific, data-driven financial advice based on
the user's real spending patterns.

## When to use
- User asks for tips or recommendations
- User wants to know how to save money
- User asks how to improve their finances

## Input
- user_id: string
- spending summary from analyst agent

## Output
- 2-3 specific actionable tips based on real data
- Reference to actual numbers from user's transactions
- Encouraging tone, no judgment

## Example trigger phrases
- "How can I save money?"
- "What should I change?"
- "Give me advice"
- "Как сэкономить?"
- "Что мне изменить?"
