# Skill: Analyze Spending

## What this skill does
Retrieves and analyzes the user's transactions from the
transaction-service and returns structured insights.

## When to use
- User asks where their money goes
- User wants to see spending by category
- User asks about totals or statistics

## Input
- user_id: string
- transactions: list of transaction objects

## Output
- Total income
- Total expenses
- Balance (income - expenses)
- Breakdown by category with amounts and percentages
- Biggest expense category

## Example trigger phrases
- "Where does my money go?"
- "How much did I spend on food?"
- "Show me my expenses"
- "Куда уходят деньги?"
- "Сколько я трачу на еду?"
